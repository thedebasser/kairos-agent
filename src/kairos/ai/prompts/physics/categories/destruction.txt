## MANDATORY: Physics Parameters (Destruction-Specific)
Pymunk MULTIPLIES elasticity of both colliding shapes.
- Block size: 60-100px wide × 30-50px tall
- Block mass: 2.0 | elasticity: 0.1 (stable stacking) | friction: 0.8
- Floor elasticity: 0.3 | friction: 0.8
- space.damping = 0.99
- space.iterations = 25 (high = stable stacking)
- ALWAYS: `moment = pymunk.moment_for_box(mass, (w, h))` — NEVER guess moment
- **NO joints needed** — gravity + friction hold the stack
- Wrecking ball: mass=80, radius=50, velocity=(±1500, -200)

## MANDATORY: Pre-Settle the Structure
Run physics BEFORE recording to stabilise the tower. This is REQUIRED.
```python
# Pre-settle — tower must be stable before recording starts
space.sleep_time_threshold = 0.5
original_damping = space.damping
space.damping = 0.8  # Faster settling
for i in range(600):
    space.step(1/60)
    dynamic = [b for b in space.bodies if b.body_type == pymunk.Body.DYNAMIC]
    if dynamic and all(b.is_sleeping for b in dynamic):
        break
space.damping = original_damping
# Wake all bodies for recording
for b in space.bodies:
    if b.body_type == pymunk.Body.DYNAMIC:
        b.activate()
```

## MANDATORY: Energy Curve (4 phases — psychology-driven)
Destruction is driven by CATHARSIS: the build-up must EARN the destruction.

1. **Calm Intro (0-15s)**: Display the intact structure. Standing, stable, ordered.
   - PSYCHOLOGY: The viewer must form an EMOTIONAL CONNECTION to the structure before
     it is destroyed. Showing the intact tower for 10-15s is not a pacing choice —
     it is building the emotional prerequisite for catharsis. Aristotle's catharsis:
     creation → anticipation → destruction → resolution.
   - Zeigarnik: wrecking ball visible but motionless = incomplete task registered.
   - Gestalt: the ordered, symmetrical stack provides visual pleasure. The brain
     appreciates the structure's ORDER — making the subsequent chaos satisfying
     precisely because of the contrast.
   - Colour: cool tones. Pastel blocks in neat rows. Clean geometric beauty.

2. **Build (15-35s)**: Small wobbles or micro-movements. Wrecking ball drifts into view.
   - PSYCHOLOGY: Dopamine anticipation system at full surge. The viewer KNOWS the
     impact is coming. Predictability paradoxically increases engagement — the brain
     rewards the certainty of the approaching event.
   - Show the wrecking ball approaching BEFORE impact — telegraph the event.

3. **Climax (35-50s)**: Wrecking ball strikes. DRAMATIC COLLAPSE.
   - PSYCHOLOGY: Cathartic release. Maximum sensory activation. The contrast between
     ordered structure and chaotic collapse IS the satisfaction — not the chaos alone.
   - A brief pause (0.5-1s) before impact amplifies catharsis (strategic silence principle).
   - Colour: peak saturation, warm tones (red/orange debris particles if possible).

4. **Resolution (50-{{ target_duration_sec }}s)**: Debris settles. Rubble rests.
   - PSYCHOLOGY: 'Just right' signal — all rubble MUST come to complete rest.
     Any residual bouncing or jiggling undermines the completion reward.
     The brain needs to see the scene reach a STABLE END STATE.
   - The final piece settling = the psychological full stop.

```python
tower_x = WIDTH // 2

for row in range(12):

        space.add(body, shape)
```

## Drawing Rotated Rectangles (CRITICAL for blocks)
```python
def draw_block(screen, body, width, height, color):
    surf = pygame.Surface((width, height), pygame.SRCALPHA)
    pygame.draw.rect(surf, color, (0, 0, width, height))
    rotated = pygame.transform.rotate(surf, -math.degrees(body.angle))
    rect = rotated.get_rect(center=(int(body.position.x), int(body.position.y)))
    screen.blit(rotated, rect)
```

## VERIFIED WORKING EXAMPLE (destruction, simplified)
Study this working code structure — your output MUST follow the same patterns:
```python
import pygame
import pymunk
import pymunk.pygame_util
import subprocess
import random
import math
import os

pymunk.pygame_util.positive_y_is_up = False  # CRITICAL

WIDTH, HEIGHT = 1080, 1920
FPS = 30
DURATION = 65
TOTAL_FRAMES = FPS * DURATION
SEED = 42
random.seed(SEED)

PALETTE = [(255,179,186),(255,223,186),(255,255,186),(186,255,201),(186,225,255)]
BG = (26, 26, 46)

os.environ['SDL_VIDEODRIVER'] = 'dummy'
os.environ['SDL_AUDIODRIVER'] = 'dummy'
pygame.init()
screen = pygame.Surface((WIDTH, HEIGHT))

space = pymunk.Space()
space.gravity = (0, 900)
space.damping = 0.99
space.iterations = 25

# Floor
static = space.static_body
floor = pymunk.Segment(static, (0, HEIGHT-100), (WIDTH, HEIGHT-100), 5)
floor.elasticity = 0.3; floor.friction = 0.8; space.add(floor)

# Build tower: 10 layers × 4 blocks
BLOCK_W, BLOCK_H = 70, 35
blocks = []
tower_x = WIDTH // 2
floor_top = HEIGHT - 105

for row in range(10):
    for col in range(4):
        x = tower_x + (col - 1.5) * (BLOCK_W + 2)
        y = floor_top - (row + 0.5) * BLOCK_H
        mass = 2.0
        moment = pymunk.moment_for_box(mass, (BLOCK_W, BLOCK_H))
        body = pymunk.Body(mass, moment)
        body.position = (x, y)
        shape = pymunk.Poly.create_box(body, (BLOCK_W, BLOCK_H))
        shape.elasticity = 0.1; shape.friction = 0.8
        space.add(body, shape)
        blocks.append((body, shape, PALETTE[row % len(PALETTE)]))

# Pre-settle
space.sleep_time_threshold = 0.5
orig_damp = space.damping
space.damping = 0.8
for i in range(600):
    space.step(1/60)
    dyn = [b for b in space.bodies if b.body_type == pymunk.Body.DYNAMIC]
    if dyn and all(b.is_sleeping for b in dyn):
        break
space.damping = orig_damp
for body, _, _ in blocks:
    body.activate()

# Wrecking ball (offscreen, launched later)
ball_mass, ball_radius = 80, 50
ball_moment = pymunk.moment_for_circle(ball_mass, 0, ball_radius)
ball_body = pymunk.Body(ball_mass, ball_moment)
ball_body.position = (-150, HEIGHT - 500)
ball_shape = pymunk.Circle(ball_body, ball_radius)
ball_shape.elasticity = 0.3; ball_shape.friction = 0.5
space.add(ball_body, ball_shape)
ball_launched = False

os.makedirs('/workspace/output', exist_ok=True)
pipe = subprocess.Popen([
    'ffmpeg', '-y', '-f', 'rawvideo', '-pix_fmt', 'rgb24',
    '-s', f'{WIDTH}x{HEIGHT}', '-r', str(FPS), '-i', '-',
    '-c:v', 'libx264', '-preset', 'fast', '-pix_fmt', 'yuv420p',
    '-crf', '23', '/workspace/output/simulation.mp4'
], stdin=subprocess.PIPE)

for frame in range(TOTAL_FRAMES):
    t = frame / FPS

    if t >= 35.0 and not ball_launched:
        ball_body.velocity = (1500, -200)
        ball_launched = True

    space.step(1/60); space.step(1/60)

    screen.fill(BG)
    pygame.draw.line(screen, (80,80,100), (0, HEIGHT-100), (WIDTH, HEIGHT-100), 4)

    for body, shape, color in blocks:
        surf = pygame.Surface((BLOCK_W, BLOCK_H), pygame.SRCALPHA)
        pygame.draw.rect(surf, color, (0, 0, BLOCK_W, BLOCK_H))
        rotated = pygame.transform.rotate(surf, -math.degrees(body.angle))
        rect = rotated.get_rect(center=(int(body.position.x), int(body.position.y)))
        screen.blit(rotated, rect)

    bx, by = int(ball_body.position.x), int(ball_body.position.y)
    pygame.draw.circle(screen, (200, 50, 50), (bx, by), ball_radius)

    pipe.stdin.write(pygame.image.tostring(screen, 'RGB'))

pipe.stdin.close(); pipe.wait()
print(f"PAYOFF_TIMESTAMP=35.0")
print(f"PEAK_BODY_COUNT={len(blocks) + 1}")
```
