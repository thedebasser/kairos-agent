# Psychology of Oddly Satisfying Content — Quick Reference

## The 7 Psychological Drivers

| # | Mechanism | Brain System | How to Activate |
|---|-----------|-------------|-----------------|
| 1 | **Completion (Zeigarnik)** | Basal ganglia / dopamine | Show "unfinished" state before action. The longer the setup, the greater the payoff. |
| 2 | **"Just Right" Signal** | Cognitive cut-off system | Simulation MUST reach unambiguous end state. All objects settled, zero jiggling. |
| 3 | **Predictability** | Ventral tegmental area / dopamine | Telegraph outcomes → confirm predictions. Viewer KNOWS what happens, not HOW. |
| 4 | **Mirror Neurons** | Supplementary motor area | Objects must feel physically real — convincing bounce, weight, momentum, tactility. |
| 5 | **Gestalt Pattern** | Visual cortex V4 / inferior temporal | Symmetrical starting arrangements, colour-coded groups, smooth continuous paths. |
| 6 | **Catharsis** | Emotional processing / cortex | Destruction must be "earned" — show order, THEN destroy. Contrast = satisfaction. |
| 7 | **Seeking/Curiosity** | Dopamine seeking circuit | Novel ARRANGEMENT of familiar physics. Not novel physics — novel configuration. |

## Satisfaction Hierarchy

| Tier | Mechanisms | Visual Elements | Design Target |
|------|-----------|----------------|---------------|
| **Tier 1** (4+) | Completion + predictability + pattern + seeking + catharsis | Cascading chains, container overflow/burst, perfect fit | **Always aim here** |
| **Tier 2** (2-3) | Mirror neurons + predictability + pattern | Bouncing/rolling, colour sorting, dramatic collapse | Acceptable |
| **Tier 3** (1) | Seeking only | Accumulation, random motion with no end state | **REJECT** |

> **Critical**: Random motion with no end state = lowest satisfaction. Every simulation needs a climax EVENT and complete resolution.

## The 4-Phase Energy Curve (Neurochemical Architecture)

| Phase | Time | Brain State | Design Rules |
|-------|------|------------|--------------|
| **Calm Intro** | 0–15s | Zeigarnik tension loading, Gestalt appreciation, seeking activation | Show complete setup. Cool colours. Ordered arrangement. Sparse sound. |
| **Building** | 15–40s | Dopamine escalation, pattern recognition, mirror neuron intensification | Each event confirms predictions → small dopamine hits. Warm colours increasing. |
| **Climax** | 40–55s | Peak reward, cathartic release, sensory activation | DRAMATIC event. Max saturation. Strategic silence before impact. |
| **Resolution** | 55–65s | Serotonin settlement, completion reward, "just right" signal | Everything settles. Cool tones return. Last event clearly audible in quiet. |

## Colour Psychology Quick Reference

### Background (60% of frame)
- `#1a1a2e` — deep navy-purple: calm, mysterious, depth (ball pits, marble runs)
- `#0d1117` — near-black blue: max contrast, cinematic (destruction)
- `#16213e` — dark steel blue: professional, clean (domino chains)

### Objects (30% of frame) — Pastel-Bright
- **Rainbow Pastel**: `(255,179,186), (255,223,186), (255,255,186), (186,255,201), (186,225,255), (212,186,255)`
- **Candy Bright**: `(224,187,228), (149,125,173), (210,145,188), (254,200,216), (255,223,211)`

### Colour Rules
- Rainbow sequencing: `colour = PALETTE[spawn_index % len(PALETTE)]` — encodes time
- Colour-code groups for Gestalt similarity (domino sections, ball layers)
- Temperature tracks emotion: cool intro → warm build → peak → cool resolve
- 60:30:10 rule: background : primary objects : accents/highlights

## Hook Caption Psychology
- Use **Zeigarnik open loops**: "What happens when..." / "Wait for the [X]..."
- Create an **incomplete task** in the viewer's mind → brain won't let go until resolved
- The 3-second rule: 50-60% of viewers leave within 3s → first frame MUST show motion

## Sound Design Principles
1. **Impact sync**: Every collision needs a matching sound, ≤150ms tolerance
2. **Material differentiation**: Hard/soft, high-freq/low-freq by material pair
3. **Density scaling**: Individual impacts (intro) → rhythmic patter (build) → textured wash (climax) → sparse final clicks (resolve)
4. **Strategic silence**: 0.5-1.0s silence before climax event amplifies catharsis
5. **Pure SFX > background music**: Music competes with ASMR mirror neuron response
6. **Rhythmic entrainment**: Regular click-click-click patterns synchronise brain oscillations

## Platform Algorithm Alignment
- **Save** (anticipation of future reward) > **Like** in algorithmic distribution
- **Share** (social bonding) > **Like** in algorithmic distribution
- **Watch-through > 100%** (replay) is the strongest positive signal
- Seamless loops exploit dopamine re-experience + seeking + completion drives
